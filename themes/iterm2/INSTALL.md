# Install (iTerm2)

1. Open iTerm2.
2. Open `Settings...` (or `Preferences...` depending on your version).
3. Go to `Profiles` -> `Colors`.
4. Click `Color Presets...` -> `Import...`.
5. Select one or both files:
   - `hanabi.itermcolors` (dark)
   - `hanabi-light.itermcolors` (dark-palette alias)
6. Click `Color Presets...` again and select the imported preset.
