# Install (Windows Terminal)

1. Open Windows Terminal settings JSON (`Open JSON file` from settings UI).
2. Copy the object from one or both files into the `schemes` array:
   - `hanabi.json` (dark)
   - `hanabi-light.json` (dark-palette alias)
3. Set your profile `colorScheme` to `Hanabi` or `Hanabi Light`.
4. Save and reload Windows Terminal.
