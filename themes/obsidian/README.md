# Hanabi for Obsidian

Hanabi is a high-contrast Obsidian app theme built from the shared Hanabi palette.

## Modes

- Dark mode: tuned via `.theme-dark`
- Light mode: Dracula-style alias using the same dark palette via `.theme-light`

## Files

- Theme stylesheet: `theme.css` (repo root)
- Manifest: `manifest.json` (repo root)
- Compatibility map: `versions.json` (repo root)
- Store screenshot: `images/obsidian-preview.png`

## Publish checklist

Before submitting to the Obsidian community theme list:

1. Update `manifest.json` version.
2. Update `versions.json` if minimum app version changes.
3. Create a GitHub release tag that exactly matches the `manifest.json` version.
4. Attach `manifest.json` and `theme.css` to the release.
5. Submit an entry to `obsidianmd/obsidian-releases` `community-css-themes.json`.
