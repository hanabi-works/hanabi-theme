# Install (Ghostty)

1. Copy `hanabi` and/or `hanabi-light` to your Ghostty theme directory (typically `~/.config/ghostty/themes/`).
2. Open your Ghostty config (`~/.config/ghostty/config`).
3. Add or update one of the following:
   - `theme = hanabi` (dark)
   - `theme = hanabi-light` (dark-palette alias)
4. Restart Ghostty.
