# Hanabi Theme for Cursor

Hanabi dark theme designed around the shared Hanabi palette.

![Hanabi Theme Preview](./images/preview.png)

## Activate

1. Open the Command Palette
2. Run `Preferences: Color Theme`
3. Select `Hanabi`
