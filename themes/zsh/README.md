# Zsh Theme

Hanabi prompt theme for Zsh.

## Files
- `hanabi.zsh-theme` - dark prompt theme file
- `hanabi-light.zsh-theme` - dark-palette alias prompt file
- `hanabi.p10k.zsh` - dark Powerlevel10k (p10k) color overlay
- `hanabi-light.p10k.zsh` - dark-palette alias p10k overlay
- `INSTALL.md` - installation instructions
